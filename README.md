# OlivePort - Ultra-Premium Olive Wood Utensils

A luxury e-commerce website showcasing handcrafted olive wood utensils with an art gallery aesthetic.

## Overview

OlivePort redefines what "style" means for e-commerce by creating an ultra-premium experience that feels like an art gallery meets a luxury culinary brand. Built with pure HTML, CSS, and JavaScript, this multi-page website showcases the beauty and craftsmanship of olive wood utensils.

## Features

### Design & Aesthetics
- **Ultra-premium styling** with deep earthy contrasts, smoky blacks, muted gold accents, and off-white stone tones
- **Art gallery aesthetic** that highlights the natural beauty of olive wood textures
- **Bespoke typography** using refined serif/sans-serif pairings (Crimson Text & Inter)
- **Asymmetrical layouts** and smooth transitions for a design studio feel
- **Cinematic full-bleed photography** showcasing products as art pieces

### Functionality
- **Multi-page architecture** with seamless navigation
- **Product catalog** featuring 10 individual utensils and 5 curated sets
- **Interactive cart system** with local storage persistence
- **Product filtering** by category (Individual Pieces vs Curated Sets)
- **Image galleries** with thumbnail navigation
- **Responsive design** optimized for all devices
- **Form handling** for contact and newsletter signup
- **Scroll animations** and micro-interactions

### Technical Implementation
- **Pure HTML/CSS/JS** - no frameworks or dependencies
- **CSS Grid & Flexbox** for modern, responsive layouts
- **CSS Custom Properties** for consistent theming
- **Intersection Observer API** for scroll animations
- **Local Storage** for cart persistence
- **Performance optimized** with lazy loading and preloading

## File Structure

```
/index.html           → Home page with hero showcase
/shop.html            → Product catalog with filtering
/product.html         → Product detail template
/about.html           → Artisan process and company story
/contact.html         → Contact form and information
/cart.html            → Shopping cart and checkout
/blog.html            → Journal/blog section

/assets/css/style.css → Complete styling system
/assets/js/main.js    → All interactions and functionality
/assets/img/          → Product and background images
/assets/fonts/        → Custom typography (if needed)
```

## Customization

The website is built with CSS custom properties, making it easy to customize:

- Colors can be modified in the `:root` section of `style.css`
- Typography scales are defined with `clamp()` for responsive sizing
- Spacing system uses consistent variables throughout
- Component styles are modular and well-organized

## Performance Features

- **Lazy loading** for images below the fold
- **Critical image preloading** for above-the-fold content
- **Optimized animations** using CSS transforms and opacity
- **Efficient JavaScript** with event delegation and throttling
- **Minimal dependencies** - only Google Fonts external dependency
